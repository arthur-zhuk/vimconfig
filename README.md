My versioned vim configuration. A collection of vim/neovim settings, plugins, and customizations for efficient text editing and development.

## Features

- **Version Controlled**: Track changes to your vim configuration
- **Plugin Management**: Organized plugin setup
- **Custom Keybindings**: Personalized shortcuts and mappings
- **Language Support**: Configurations for various programming languages

## Installation

```bash
git clone https://github.com/arthur-zhuk/vimconfig.git ~/.vim
cd ~/.vim
```

Or for Neovim:

```bash
git clone https://github.com/arthur-zhuk/vimconfig.git ~/.config/nvim
cd ~/.config/nvim
```

## Usage

After cloning, vim/neovim will automatically use the configuration files. Restart your editor to apply changes.

## Configuration Files

- `.vimrc` or `init.vim` - Main configuration file
- Plugin configurations
- Custom key mappings
- Language-specific settings
